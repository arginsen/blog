银行系统软开 | IBM MQ 接收报文



IBM MQ 是 IBM 提供的一种 企业级消息中间件，广泛用于金融、银行等对稳定性和安全性要求高的系统中。它支持 JMS（Java Message Service）标准，可以与 Java 程序很好地集成。

IBM MQ 层级

名称 描述
Queue Manager 队列管理器，MQ 的核心组件
Queue 队列，消息的存储通道
Channel 通道，客户端与队列管理器通信的方式
Listener 监听器，监听某个端口接收连接请求
Message 消息内容
